# react-feed
## Simle react app with VK like feed

## Features:
 - Authorization
 - Likes
 - Comments
 - Ability to display and create new posts
 
 ## Tech:
 - React
 - Prisma 2 (ORM)
 - Tailwindcss (styling)
 - Express (for rest)
